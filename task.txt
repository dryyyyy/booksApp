Учет частоты слов в библиотеке.
Задача:
Есть директория, в которой лежат книги в файлах формата *.txt. Книги добавляются и
удаляются из директории не через веб-интерфейс и не чаще одного раза в 10 секунд.
Количество книг большое.
Необходимо спроектировать и реализовать систему по сбору и хранению статистики
разных видов, а также api для их получения.
Виды статистики:
1. Кол-во уникальных слов в заданной книге.
2. Частота вхождения заданного слова в заданной книге.

Условия:
Уникальным словом можно считать любую последовательность символов между
символами разделителями - “ ”, “,”, “.”, “;”, “:”.
Частота вхождения слова - это отношение кол-ва вхождений слова к общему кол-ву
слов.
Реализация должна быть на php либо php + golang. В остальном ограничений на
используемые технологии нет.
Должен быть файл README.md, который содержит информацию необходимую для
сборки, запуска и эксплуатации приложения.
